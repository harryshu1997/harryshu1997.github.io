Using Claude to parse papers & chat?

After signing in, open the Papers tab and fill the Claude card. Two ways to power it:

A) Anthropic API key — paste an sk-ant-… key, leave Base URL blank. Billed per token at console.anthropic.com (separate from a Claude.ai subscription).

B) Claude.ai subscription via CLIProxyAPI — to avoid per-token billing:

  1. Install: brew install cliproxyapi (macOS), or use the Linux one-liner / Docker from the docs.
  2. Log in with your Claude account: run the proxy's Claude login (e.g. cli-proxy-api --login) and authorize in the browser — this saves your Pro/Max OAuth token. Exact command: see docs link below.
  3. Start it: brew services start cliproxyapi — serves http://127.0.0.1:8317.
  4. Add an api-keys entry in config.yaml.
  5. In the Claude card above: API Key = that key, Base URL = http://127.0.0.1:8317 → Save.

CLIProxyAPI docs & exact login command ↗

CLIProxyAPI only works while running on this machine, so option B is local-only.
